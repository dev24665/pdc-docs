    Aliquot Run Metadata

    Summary
    Type Aliquot Run Metadata
    Category Metadata
    Description
    Experimental metadata describing how an aliquot was processed within a study.
    Unique Keys
    • aliquot_run_metadata_id

    Links
    Name Required Label
    Study Run Metadata True Aliquot Run Metadata references the experimental run of the study.
    Aliquot True Aliquot Run Metadata references the Aliquots used in an experimental run of the study.


    Properties
    Name Description Acceptable Types/Values Required CDE
    acquisition_file_name Representative file name for one of the raw data file from the run String False -
    analyte Type of peptide or protein enrichment String False -
    condition Biological condition (ex: disease state) of the cases(patients) from which the aliquots derived. String False -
    date Date of the data acquired from the mass spectrometry instrument String False -
    experiment_number Number associated with a specific run of the experiment String False -
    experiment_type General strategy used for differential analysis String False -
    folder_name Name used to group the files generated in single experimental run String False -
    fraction Number of fractions in the experimental run String False -
    instrument Make and model of the mass spectrometry instrument String False -
    label Reagent used for isobaric labeling or label-free String False -
    operator Mass Spectrometry instrument operator String False -
    protocol Name of the protocol used to run the experiment String False -
    replicate_number Replicate number, if applicable String False -
